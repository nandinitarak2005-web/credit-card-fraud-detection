# 💳 Credit Card Fraud Detection

## 🚀 Overview
This project uses Machine Learning to detect fraudulent credit card transactions.

## 🧠 Model Used
- Random Forest Classifier

## 📊 Features
- Data preprocessing
- Model training
- Fraud prediction
- Confusion matrix visualization

## 🛠️ Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Google Colab

## 📈 Results
The model successfully classifies fraud and normal transactions with good accuracy.
